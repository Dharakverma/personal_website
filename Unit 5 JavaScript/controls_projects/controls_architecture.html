The goal of the Controls System project is to to design, simulate, implement, and test a vehicle control system for the 2022-2023 MAC Formula Electric vehicle. The control system will allow the vehicle to be operated in basic driving conditions by managing the following vehicle subsystems: battery management, cooling, vehicle mode selection, tractive motor, and vehicle dynamics. This project is specific is purposfully abstracted away from the embedded system (as much as possible) to simplify the black box, white box, and unit testing that must occur before it is ready for implementation on the vehicle. The control system will be developed using MATLAB and Simulink, in a model-based design. The models will be converted to C code using the relevant Simulink toolkit and then handed off to the Embedded Software team for integration onto the relevant microcontroller.

The following table provides brief explinations on each of the control system blocks:

Control Block Overview
Governor The super-state machine of the entire vech icle. Computes current vehicle status, recieves updates from other blocks, and sends commands accordingly.
Torque Path Controller Contains all required logic to command and interface with the propulsion hardware for the vehicle, including the pedal and steering sensors and motor controllers.
Battery Monitor Contains all required logic to compute the current status of the battery pack for the vehicle. Sends the relevant status to the Governor to affect vehicle status and state.
